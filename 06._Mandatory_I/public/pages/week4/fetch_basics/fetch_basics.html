JavaScript Fetch API

The Fetch API interface allows web browser to make HTTP requests to web servers.

A Fetch API Example:


                        // In simple, this example fetches a file and displays the content:
                        fetch(file)
                        .then(x => x.text())
                        .then(y => myDisplay(y));

                        // Since Fetch is based on async and await, the example might be easier to understand like this:
                        async function getText(file) {
                            let myObject = await fetch(file);
                            let myText = await myObject.text();
                            myDisplay(myText);
                          }
