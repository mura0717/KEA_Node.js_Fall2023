Structuring HTML with Semantic HTML

HTML is the standard markup language for creating Web pages.
It consists of elements that tells the browser how to display them.

With over 100 HTML elements, and the ability to create custom elements,
there are infinite ways to mark up your content; but some ways -semanticallyâ€” are better than others.

Semantic means "relating to meaning". Writing semantic HTML means using HTML elements
to structure your content based on each element's meaning, not its appearance.

An exampel code with elements without semantic value:

Same code - rewritten with semantic elements:

              one word
              one word
            

          

          

            

              
five words

            

            

              
three words

              
forty-six words

As seen in the second code block, semantic elements allow the HTML structure
to be understood even without understanding the content, as they also provide meaning.
