Package.json

A package.json is a JSON file that exists at the root of a Javascript/Node project.
It holds metadata relevant to the project and it is used for managing the project's dependencies, scripts, version, etc.

Example of package.json file:


    {
        "name": "my-nodejs-project",
        "version": "1.0.0",
        "description": "A sample Node.js project",
        "main": "index.js",
        "scripts": {
            "start": "node index.js",
            "test": "mocha tests/*.js"
        },
        "author": "Your Name",
        "license": "MIT",
        "dependencies": {
            "express": "^4.17.1",
        }
    }

The following command will install the project dependencies once executed in the folder where package.json is located at.


        npm install
