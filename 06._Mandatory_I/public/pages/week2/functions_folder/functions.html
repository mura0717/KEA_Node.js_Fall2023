Functions in JS

A JavaScript function is a block of code designed to perform a particular task.

Types of Functions:

  • Normal Functions: Functions that have an arguments object which you can access in the function:

    
                                function greet(){
                                    console.log("Hello!")
                                }

  • Arrow Functions: Anonymous functions without a name and are not bound by an identifier. They do not return any value and can be declared without the function keyword. 

    
                                const greet = () => {
                                    console.log( "Hello!" );
                                }

  • Anonymous Functions: It is a function that does not have any name associated with it.

    
                                const greet = () => {
                                    console.log("Hello!");
                                };

  • Callback Functions: A callback is a function that is passed as an argument to another function, and is called after the main function has finished its execution. 

    
                                function square(number, callback) {
                                    const result = number * number;
                                    callback(result); // Call the callback with the squared result
                                    }
                                    
                                    function displayResult(value) {
                                    console.log(`The result is: ${value}`);
                                    }
                                    
                                    // Call the square function with a callback
                                    square(5, displayResult);
