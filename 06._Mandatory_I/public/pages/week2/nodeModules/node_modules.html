Node Modules

In Node.js, Modules are the blocks of encapsulated code that communicate with an external application on the basis of their related functionality.
Modules can be a single file or a collection of multiple files/folders.

Types of Modules:

  • Core Modules: Built-in modules that are part of the platform.
  • Local Modules: Modules that are created locally in your application.
  • Third-party Modules: Modules that are available online using the Node Package Manager(NPM).

Modules can be loaded into the program by using the required function.


        const module = require('module_name');
        
        // For example, for the Express module:
        const express = require("express")
        const app = express();
