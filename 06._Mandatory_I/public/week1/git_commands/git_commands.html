Basic Git Commands

Install NodeJS

            $ sudo apt-get install -y nodejs
            
            $ apt-get install -y nodejs
Check NodeJS Version

            $ node -v
            
            $ npm -v
Install & Uninstall a Package

            $ npm install <package>

            $ npm uninstall <package>
Create an Empty 'npm' Project & Initialize a New 'package.json' File

            $ npm init -y 
            {
              "name": "",
              "version": "",
              "description": "",
              "main": "",
              "scripts": {
                "test": "echo \"Error: no test specified\" && exit 1"
              },
              "keywords": [],
              "author": "",
              "license": "",
              "dependencies": {}
            }
List Modules Locally & Globally

            $ npm ls

            $ npm ls -g
Run Node.js script

            $ node <app_file>
