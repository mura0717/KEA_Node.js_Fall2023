What are Functions?

A JavaScript function is a block of code designed to perform a particular task.

Types of Functions:

  • Normal Functions:
    This is the standard method of writing a function. 
    
                    function greet(){
                        console.log("Hello World")
                    }
  • Arrow Functions:
    Arrow functions are anonymous functions i.e. functions without a name and are not bound by an identifier. Arrow functions do not return any value and can be declared without the function keyword.
